# 【ptt alertor】規格計劃書

## 專案目標

即時監控 PTT 指定看板的文章與回文，符合條件時主動推播通知。
## 目標受眾

- 金融投資人（股票、加密貨幣討論區）
- 研究人員（學術看板監測）
- 品牌公關（輿情監控）

差異化：多平台推播 + 彈性關鍵字邏輯 + 友好的設定 UI。
## 功能規格

### 看板管理

- 新增 / 刪除 / 編輯監控看板
- 每看板獨立設定
### 關鍵字規則引擎

- 支援 AND / OR / NOT 邏輯
- 支援正規表達式
- 排除清單（噪音詞）
### 推播設定

- Discord Webhook
- LINE Notify
- Email（SMTP）
- 推播頻率限制（防轰炸）
### 歷史記錄

- 符合條件的文章列表
- 點擊率追蹤
## 技術建議（給 Alan）

- 爬蟲：Python + requests + BeautifulSoup（Docker Compose 部署）
- 排程：Celery + Redis（每 30 秒檢查）
- 前端：Next.js（設定 UI）+ Tailwind
- 後端：FastAPI
- 儲存：PostgreSQL（規則 + 歷史）
